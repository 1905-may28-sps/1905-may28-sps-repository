CSS Box Model

The CSS Box Model is a set of rules which determine the dimensions of every element in a web page. It gives each box (both inline and block) four properties which define the space they take up.

  • Content- the text, image, or other media content in the element
  • Padding- this space between the box's content and its border
  • Border - the line between the box's padding and margin
  • Margin- the space between the box and its surrounding boxes
