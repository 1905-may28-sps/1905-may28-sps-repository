Understanding HTTP requests with Angular


Most front-end applications communicate with backend services over HTTP. One common approach to asynchronous requests and responses with Angular is to use Observables and the HttpClientModule

So what are Observables?


Obserables are arrays whose items arrive asynchronously over time. Obserables help you manage asynchronous data, such as data coming from a backend service. Obserables are used within Angular iself, including its event system and http client service. To use observables, Angular uses a third-party library called Reactive Extensions (RxJs).


When we send HTTP requests, we usually put the code in Angular Services
