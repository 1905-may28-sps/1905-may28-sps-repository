Understanding Directives

Angular Directives are special pre-built instructions that typically render HTML code or provide additional functionality

There are three types of directives:

  • Custom Directives using the decorator @Directive. - This is added to a TypeScript class. Furthermore, The @Component decorator is actually a special form of the @Directive decorator. In practice, you’ll use @Directive when creating your own custom directives.
  • Structural Directives - This will add/remove/or update the contents of the DOM. It basically changes the webpage’s layout. *ngFor and *ngIf which we’ll explain in a moment are common examples of structural directives.
  • Attribute Directives - The final form is an attribute directive. An attribute directive will change the look and feel or behavior of an element. Examples of attribute directives are ngModel, ngClass, and ngStyle. We cover ngModel in a separate video when we explain two-way data-binding. Here, you’ll see usage of ngClass.

*ngFor
