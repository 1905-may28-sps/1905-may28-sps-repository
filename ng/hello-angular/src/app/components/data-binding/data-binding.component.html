About Data Binding

Angular supports data binding, which is a mechanism for coordinating parts of a template(HTML) with parts of a component (TS). We no longer use DOM element selection with angular, we simply add binding markup to our template HTML which will tell angular how to connect both sides

Interpolation

Interpolation is a way to display data from components in our templates, we can also use the same syntax for template expressions A few examples:
  • 2 + 2 = {{ 2 + 2 }}
  • text from component: {{ text }}
  • Count = {{ count }}
  • More math: {{19849823/9}}
  • Using Math.random in TS file: {{someNum}}
  • Expressions : {{ count > 0 ? 'count is greater than 0' : 'no its not'}}
Some rules about template expressions, they cannot be used with the following:
  • assignment operators (=, +=, -=, etc)
  • operators such as new, typeof, instanceof
  • chaining expressions with ; or ,
  • increment (++) or decrement (--)
  • no support for | &
