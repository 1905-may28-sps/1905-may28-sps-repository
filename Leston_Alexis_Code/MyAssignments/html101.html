Some tags that you may need to know for more enriching applications

  • The abbr tag: this represents the abbreviation element which helps provide more info on some text
  • Iframes are elements that allow you to bring in a view of an entire web page in your site
  • A hyperlink , one of the most important types of elements, allows you to link another page to your page. With hyperlinks, we can click text, buttons, or images and open another site. In order to use hyperlinks, we use the a tag which is actually the anchor tag with an attribute of href. The Anchor tag simply takes you to another location, for example the top of the page
  • Images allow us to add pictures (and gifs!) to our HTML page. The img tag uses an src attribute for the source of your media (can be local or can be from the internet) and the alt attribute which holds the text alternative for an image. There are also other attributes you can add such as width, height, border, etc.
