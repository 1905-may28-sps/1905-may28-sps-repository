Simple Storage Service

Amazon S3 or Simple Storage Service is a object storage built to store and retrieve any amount of data from anywhere - web sites, mobile apps, corporate apps, and from IoT sensors or devices


We can use S3 for various things. For example, we will host this very application on the cloud via an S3 bucket!
How does one do this? Thanks for asking....
  • Create S3 bucket with public accessiblity
  • In your S3 Management Console, click properties, then select Static Website Hosting
  • Select 'Use this Bucket to host a website'
  • Denote your index.html as both your Index Document and your Error Document
  • Back in your Angular app, cd into your project folder and run the command ng build --prod
  • This will package your applicaiton into a dist folder inside the project
  • Copy or upload the contents of this dist folder into your S3 bucket, making the content also publicly accessible
  • Navigate to the endpoint URL for your site denoted in the properties tab where you have selected to use your S3 bucket for static website
