the css box model

the css box model is a set of rules which determine the dimensions of every element in a web page. it gives each box both inline and block four properties which define the space they take up.

  • content- The text,image or other media in element
  • padding- The space between the boxs content and its border
  • border- The line between the boxs paddinmg and margin
  • margin- The space between the box and surrounding boxes
