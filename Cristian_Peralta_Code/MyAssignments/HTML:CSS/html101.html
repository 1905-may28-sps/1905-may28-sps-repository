Some tags that you may use to enrich applications

  • The abbr tag
  • Iframes are elements that allow you to bring in a view of an entire web page in your site
  • A hyperlink, one of the most important types of elements, allows you to link another page to your page. with hyperlinks, we click text, buttons or images and open another site. In order to use hyperlinks,we use the a tag which is actually the anchor tag with an attribute of href. The anchor tag simply takes you to another location
  • Images allow us to add pictures and gifs to our html page The img tag uses an src attribute for the sourcse of the media can be local or from the internet and the alt attribute holds text alternative for an image. There are also other attributes you can add such as width, height, border, etc.
  • The div element is used to group multiple elemetns together to apply styling or javascript. the div element itself has no stylistic or behavioral properties other than to serve as a grouping for other elements.
  • There also exists the span element which is essentially an inline version of div. This allows us to group a particular line and add styling or js to it
